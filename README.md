## Function list

1. Make Windows 10 taskbar transparent/translucent

2. Make Windows 10 desktop invisible/visible

3. Make Windows 10 startmenu invisible/visible

4. Make Windows 10 explorer window transparent/translucent

5. Execute programs

6. Terminate program
